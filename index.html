《the sexy brutale》

发表于 2022-03-05 更新于 2022-03-06 分类于 杂谈，行车记录仪 阅读次数： Valine：
本文字数： 1.5k 阅读时长 ≈ 1 分钟

有个人推荐了艾迪芬奇的回忆，所以突然想起来去找点剧情游戏玩，b站上搜了一下，加上还能买到$cdkey$的游戏也没有多少，就入了这个。而且也是这个时间回溯解密的噱头很吸引我。

这游戏流程比较短，玩了一晚上8个小时就通了。就第一感觉来说，没有很惊艳，如果打分的话，8分比较合适。比及格高一个档次，主要的问题在于解谜要素比较浅，有些关莫名其妙地就过了，像是酒吧关，还没明白怎么回事，开了摄像头就过了，还有舞台的两个拉杆，我都没有去听凶手的话就猜到了要两边一起拉。

另一方面，作为一个剧情游戏，它剧情有点单薄，开始听到广播八点有表演，舞台下魔术的道具笼子，12点之前下起的血雨，我还以为后面会有对应的剧情，结果像是埋的伏笔被放弃了，我觉得完全可以加一些。还有就是最后的反转有点强行，有点为了转而转，以想象来解释发生过的超自然现象我不太喜欢，有种作者编不下去了的感觉。不过也看得出来为了这个反转游戏在前期藏了很多彩蛋，像是扑克，酒吧，主人公的名字（虽然我都没看出来）。

聊到这里，突然意识到这游戏除了剧情有点伤其它方面都做的很好。无论是新颖的玩法（不那么新，不过比换皮游戏好），还是精致的伪3D画面。个人最喜欢的是音乐室，凶手弹的钢琴真的特别好听，加上死者的歌喉，有种区别于整个游戏紧迫主色调的体验。很多地方的细节也做的让人称赞，尾声前的那个挂画房间一幅一幅的灭掉的设计很有意思，让我想起来柯南《贝克街的亡灵》里面那个大本钟。游戏进行中的音效也对应着一些角色的死亡，最醒目的枪声的和钟声，在见证死亡的时候会豁然开朗：原来是这样；继续进行游戏的时候每次听到这些声音就会想起：这个时候谁该死了。

总的来说剧情过于简单了，但是在细节上做的很好，如果把剧情拉长，讲一些人物背景而不是用物品的文字描述来表现，多增加一些场景间的关联会更好。因为我个人很喜欢这种碎片化的剧情。

最后谈一谈整个游戏最后的结局。真相非常简单，主人公想结束自己“放浪”的生活，决定烧掉自己的豪宅骗保，结果定时装置出错失手烧死了自己的一众友人以及妻子孩子，为了惩罚自己用思想建造了“sexy brutale”，看着大家的死亡来折磨自己。处于某种原因，自己或者是“妻子的灵魂”想结束这一切，就派了“另一个自己”来探求真相，劝一下自己。

其实这个问题一直是一个永远令人纠结的问题，在真正的生活中其实也有很多这种事。不可调和的矛盾点在于站在受害者的角度来看凶手是不可原谅的，而这种情绪持续而不消减反而成了另一种暴力。站在另一种角度来说也许惩罚过重，或者说不应该受到这种惩罚。虽然说以暴制暴没什么错，但是老是卡在这儿也不好。即使是有法可循的都不能抚平受害者的感受，更何况那些无法可依的东西。游戏里有一句话说的很好：

我们不是来原谅你的，我们永远做不到。

原谅究竟代表什么？

如果不能原谅，为什么又来拯救？

一开始觉得矛盾，后来有些理解了。做错了事就要接受惩罚，这是我一向的观点。从客观的角度来说错误有大有小，代价有大有小，大小相互对应十分重要，法律就是一个很好的例子。但是对应是很难的，不论是感性还是理性都没有一个确切解。又回到了最初的矛盾点，究竟到什么时候到什么程度，双方立场多数是不同的，这个分寸往往落到某个人手里，在没有制约的情况下，又往往是自己。个人的价值观就十分重要了，原谅与否是每个人自己的选择，能不能原谅自己是自己决定的，这么说也许有点自私，但是对于拥有良知的人是最好的。

可以这么说，错误都是无法挽回的，与其说乞求原谅，不如付出代价再继续向前走。毕竟每个人都有理由被尊重，就像每件事都有自己的缘由。不管怎么说，正在进行的和将要进行的事情都比已经过去的有意义，这并不是出于谁对谁的谅解，而是出于这个世界的最优解。

本文字数： 2.3k 阅读时长 ≈ 2 分钟

校历第五周。

情节性很强，两天就读完了。读着很令人伤感，在文字中体会到了南京大屠杀的恐怖。

女学生们虽然敌视“女人们”，但最后还是只能接受她们的保护。在战争人们的本性也都暴露出来，不管以前身份的高低贵贱。英格曼神甫在最后鬼子来要女学生的时候竟然想到将“女人们”送出去，我想这不是一个神甫的做法。相比之下法比就有骨气的多，痛骂了英格曼神甫一顿。在书中军人们和赵玉墨都是很有牺牲精神的，但是身份却是所有人当中“生命最下贱”的。大家会想：士兵死了就死了，战争中就是该死的；妓女死了就死了，反正活着也是祸害别人的家庭。有些时候我们竟然觉得这理所应当，这未免太骇人了。

读《上种红菱下种藕》

发表于 2021-03-27 分类于 杂谈 ， 读书笔记 阅读次数： Valine：
本文字数： 1.8k 阅读时长 ≈ 2 分钟

校历第三周。

着实感觉上课看书效率有些低，只能期待习概和C语言赶紧结课吧。

初见这个标题感觉很有诗意，就选择读它了。虽然实际内容与想象中有些不符，但是读完之后，我确实很喜欢它。

我觉得说王安忆继承的张爱玲的风格是不无道理的，书中穿插了很多水乡小镇的描写，有些炫技之感，喜欢的人会觉得好，不喜欢的人也会很反感。也有人讲张炫技，我只读过张的中短篇，里面的句子很华丽，但是没有那么多大篇幅的描写，有些让人印象很深。个人不大喜欢大量描写，所以觉得张恰到好处而王有些冗杂了。不过正因如此，水中飘着的塑料泡沫，山墙上的绿苔裂痕，以工笔描绘的华舍镇得以展现在眼前。

读《遍地枭雄》

发表于 2021-03-19 更新于 2021-03-27 分类于 杂谈 ， 读书笔记 阅读次数： Valine：
本文字数： 1.6k 阅读时长 ≈ 1 分钟

校历第二周。

开学季闲来没有什么事，读书的速度便快一些。不过近代史、C语言之类的课看书容易走神，本来半天读完的书，要读十几节课。淘宝上一本几块钱的书，买了王安忆的五本，可惜没有《长恨歌》。说实话，纸质的阅读体验确实要好很多。

这本书大致可以分为两个部分，主人公前面叫韩燕来后面叫毛豆。韩燕来在家里被姐姐宠的要死，没怎么接触社会，导致了他天真单纯的性格。在常人眼里日本工厂领着高工资，舒适的工作环境，他却由着性子辞掉了，商量着和朋友一同创业，在朋友们在现实的压迫下作鸟兽散以后，韩燕来又只能在家里待业，之后转去本土工厂上班。那时候上海想来也是一个发展极其迅速的城市，只要有心，什么工作都不差。可是韩燕来却恋家，表叔的木匠手艺也不愿意离家去学，怎么看都是个不会做打算的小孩子，工厂倒闭后众人去讨薪也就凑个热闹，似乎不在意薪水发不到自己头上。最后还是姐姐出钱让韩燕来学车，他才当上了出租车司机，又有能力养活自己。

读《欧也妮·葛朗台》

发表于 2021-03-10 更新于 2021-03-27 分类于 杂谈 ， 读书笔记 阅读次数： Valine：
本文字数： 1.3k 阅读时长 ≈ 1 分钟

开年来读的第一本书。寒假没日没夜地守在电脑旁边，说起来感觉也有些虚度，这本书本来计划在上学的火车上看的，不过车上睡过去了一半，醒着就刷手机，一页也没动。最后拖到了三月份才读完这本薄薄的书。

读完让人印象最深刻的是葛朗台老头这个标准的吝啬鬼。这种形象在巴尔扎克的小说中似乎屡见不鲜，他们将拥有的财产作为生命的价值。商人之间的利益联姻想起来，在中国古典名著中似乎没有类似的描写，而是政治联姻，可能是因为没有资本主义社会的财产丰富？抑或是封建制度中商人极低的地位。不过本质都是一样的，无论时间定格在哪一个年代，感情为财富服务的故事数不胜数。
