读《金陵十三钗》

发表于 2021-04-11 分类于 杂谈 ， 读书笔记 阅读次数： Valine：
本文字数： 2.3k 阅读时长 ≈ 2 分钟

校历第五周。

情节性很强，两天就读完了。读着很令人伤感，在文字中体会到了南京大屠杀的恐怖。

女学生们虽然敌视“女人们”，但最后还是只能接受她们的保护。在战争人们的本性也都暴露出来，不管以前身份的高低贵贱。英格曼神甫在最后鬼子来要女学生的时候竟然想到将“女人们”送出去，我想这不是一个神甫的做法。相比之下法比就有骨气的多，痛骂了英格曼神甫一顿。在书中军人们和赵玉墨都是很有牺牲精神的，但是身份却是所有人当中“生命最下贱”的。大家会想：士兵死了就死了，战争中就是该死的；妓女死了就死了，反正活着也是祸害别人的家庭。有些时候我们竟然觉得这理所应当，这未免太骇人了。

英格曼神甫这个人是很矛盾的。神甫本来以为教堂日本人是不会进来的，在学生家长的嘱托下，保护着女学生。在我看来，自从他的衣服被日本军刀划破之后，他就害怕了，他没有勇气在日本军队面前保护女学生们。客观来说他也没有能力去保护。而且他的心目中的价值观是有问题的，在他的眼中妓女们是吸学生们的血的，而且自己没有义务保护妓女，所以果断出卖了妓女们：

他被迫做出这个选择，把不太纯的、次一等的生命择出来，奉上牺牲祭台，以保有那更纯的、更值得保存的生命。

他要向她们表示痛心：事情只能这样子，日本人带走的只能是她们。只能牺牲她们，才能搭救女孩们。

书中也提到“神甫仍然不尊重她们，但不再嫌恶她们”。所以至始至终，神甫都没有平等的看待生命。这应该不是基督推崇的吧。

法比虽然和英格曼神甫站在同一个立场上，但是我认为他最后应该是消除了对妓女们的偏见的。在和赵玉墨谈了心之后，法比是明白了生命的平等。他与英格曼神甫的不同点在于他操着扬州话，是个中国人。我想这就是他要保护妓女们的原因。

“快到地下室去！”法比对女人们说：“我活着，谁也别想拉你们做垫背的！”

军人们是大义凛然的，戴涛也因为自己的信念和神甫争吵了几番。可惜他们被汉奸出卖了，尽管实际上并没有认出真正从江边逃跑的，但是鬼子是不讲理的，连手无寸铁的兵也杀，但是他们是死得硬气的，值得尊敬的。另外戴涛和赵玉墨的关系是很有意思的，虽然身份相差悬殊，但是在都不怎么被教堂接纳的情况下，颇有些“同是天涯沦落人”的意味。大概也是戴涛对赵玉墨心动的原因吧。

赵玉墨是不同于其他“女人”的，在初到教堂，大家矛盾异常激烈的时候，总是要靠玉墨来拉架。说实话甚至感觉赵玉墨不像是当娼那样粗俗的女人。玉墨给戴涛讲的故事，我想也是半真半假的。她成为妓女也是无奈的。在玉墨身上真的是一段悲惨的人生，正如她在戴涛墓前同法比讲的那样：

哪怕就是用这下贱营生，她也要出人头地。

即便如此，玉墨非同一般的伟大，在听到神甫说要交出学生的时候立刻就明白了是什么意思，也没有让神甫说出他心里想的。她号召姐妹们代替女学生们去死：

玉墨恼怒地说：“好，有种你们就在这里藏到底，占人家地盘，吃人家口粮，看着日本人把那些小丫头拖走去祸害！你们藏着是要留给谁呀？留着有人疼有人爱吗？”她现在像个泼辣的村妇，一句话出口，好几头挨骂，但又不能确定她究竟骂谁。“藏着吧，藏到转世投胎，投个好胎，也做女学生，让命贱的来给你们狗日的垫背！”

这就是玉墨“出人头地”的方式。

最后讲一讲豆蔻吧。那一段差点把我看哭了。

豆蔻给王浦生上了药，把她的琵琶抱起来。琵琶只剩一根弦，最粗的那根，因而音色低沉浑厚。豆蔻边弹边哼，过一会问王浦生：“好听吗？”

“好听。”王浦生说。

“真好听？”

“嗯。”

“以后天天给你弹。”

“谢谢你……”

豆蔻说：“不要谢我，娶我吧。”

这回没人拿她当傻大姐笑。

“我跟你回家种田。”豆蔻说，小孩过家家似的。

“我家没田。”王浦生笑笑。

“你家有什么呀？”

“……我家什么也没有。”

“……那我就天天给你弹琵琶。我弹琵琶，你拉个棍，要饭，给你妈吃。”豆蔻说，心里一片甜美梦境。

“我没妈。”

豆蔻愣一下，双手抱住王浦生，过一会，人们发现她肩膀在动。豆蔻是头一次像大姑娘一样哭。

原先在傻哭的呢喃，此刻陪着豆蔻静静地哭。周围几个女人都静静地哭起来。

豆蔻哭了一会，拿起琵琶一摔：“都是它不好！把人都听哭了！就这一根弦，比弹棉花还难听！”

其实豆蔻是“女人们”中最像女学生的，小孩子气，真心。

豆蔻在七岁就是个绝代小美人，属于心不灵口不巧心气也不高的女子，学个发式都懒得费事，打牌输了赌气，赢了逼债，做了一年，客人都是脚夫厨子下等士兵之流。挨了五年打，总算学会了弹琵琶。身上穿的都是姐妹们赏的，没一件合身，还有补丁。妓院妈妈说她：“豆蔻啊，你就会吃！”她一点不觉得屈得慌，立刻说：“唉，我就会吃。”她唯一长处是和谁对路就巴心巴肝伺候人家。

她若想巴结谁就说：“我俩是老乡吔！”所以普天下人都是豆蔻的老乡。她若想从客人或者姐妹那儿讨礼物，就说：“哎哟，都搞忘了，今天是我生日哎！”所以三百六十五天都可能是豆蔻的生日。

这里说什么感觉也不合适，豆蔻死的那一段太残忍了，就不引了。豆蔻也是被卖到窑子里的。难道生命就非要分什么贵贱吗？网上有很多质疑这本书的声音，讲什么书里宣扬把生命分成三六九等不好。我觉得这个故事不是讲的这个，我更觉得这是一些人舍弃生命救另一些人的故事。女学生我是一点好感都没有的，屁都不懂一个，等着狗日的命贱的来垫背。可是这不是我们大多数人的写照吗？自认为高尚，实际上下贱得要死。

战争真的是世界上最为恐怖的事，所到之处横尸遍野，遍地焦土。无休止的战争与无休止的杀戮真是人类文明里的悲哀。中国一定要强大起来，保护好我们的人民不落入这样的禽兽之手。战争带来的伤痛对于一个人来说是不可磨灭的，正如文章最后法比提到的：

他说也许赵玉墨只能成为另一个人才能活下去。

窑姐们的简单愿望因为命运怎么做都实现不了，还要在灾难到来之时先去赴死，一半无奈，一半慷慨。

这个世界平等是很难的了，但是起码尊重我想大家应该都是能做到的。

补《RE0》

发表于 2021-04-06 分类于 杂谈 ， 补番小记 阅读次数： Valine：
本文字数： 632 阅读时长 ≈ 1 分钟

RE0终于出了第二季了，似乎距离第一季已经4年了？有点记不清了。

听说白狐社第二季经费堪忧，但是以肉眼可见的诚意收获了巨大的好评。每集29分钟左右，而且经常oped同时消失（话说感觉都还挺好听）。b站评分从9.8跃至9.9也印证了只要用心做，良好反馈是肉眼可见的。（希望在日本不要暴死）

看第二季的时候基本上把第一季剧情忘得一干二净，边看第二季边回忆起来了点。在我的心目中，第一季的剧情还是优于第二季的。我在第二季的发展有点不解（骡子和贝蒂到底想干啥，反正我完全没有同感，可能是没理解到点）。感觉看轻小说改都有一点这个问题，剧情感觉不是很连贯。

不过除去这个来说，场面做得很不错，486多次回跳也很扣人心弦。剧情很吸引人，我一天刷个七八集不到五天就刷完了。前段时间看龙与虎磨蹭了十集就没动力了，似乎好久没追过这种一天十集的番了。可惜我不喜欢艾米莉亚，感觉太作了，试炼过N遍都过不去，也不信任帕克和486，还要等486来“话疗”。拉姆雷姆强欲哪个不比艾米莉亚好？所有角色里最不喜欢艾米莉亚。关键是还把雷姆干翻了。（雷姆yyds！！！）

从“拉姆已经够可爱了”开始喜欢拉姆。喜欢拉姆雷姆这样的性格，够独立，也有能力。

现在回忆起第一季似乎剧情流畅许多（白鲸后面有点忘了），起码前面在骡子家里的时候按部就班，刚刚看的时候一集死一次算是惊心动魄了，总想点开下一集去一探究竟。

总的来说RE0是异世界番里的巅峰了，有悬疑、友情、成长、战斗，不像某些穿越番一直爽，还是喜欢一些有想法的番。

校历第四周。

这一周读的比前几周的长，周六下午还花了点时间去看。《长恨歌》中经历的时间跨度太长了，人物出场也比较多，所以读书笔记感觉会比较乱。其实这段时间读了很多书，感觉对于文字的理解还是很浅显，或是阅历不够，或是思想不够深入。总之越来越觉得自己懂得太少了，而且上了大学以后感觉时间过得前所未有地快，在高三的时候都是掰着指头算的，自己的生命也即将过去二十年，想想有点恐怖。想学的东西还有很多。

读《上种红菱下种藕》

发表于 2021-03-27 分类于 杂谈 ， 读书笔记 阅读次数： Valine：
本文字数： 1.8k 阅读时长 ≈ 2 分钟

校历第三周。

着实感觉上课看书效率有些低，只能期待习概和C语言赶紧结课吧。

初见这个标题感觉很有诗意，就选择读它了。虽然实际内容与想象中有些不符，但是读完之后，我确实很喜欢它。

我觉得说王安忆继承的张爱玲的风格是不无道理的，书中穿插了很多水乡小镇的描写，有些炫技之感，喜欢的人会觉得好，不喜欢的人也会很反感。也有人讲张炫技，我只读过张的中短篇，里面的句子很华丽，但是没有那么多大篇幅的描写，有些让人印象很深。个人不大喜欢大量描写，所以觉得张恰到好处而王有些冗杂了。不过正因如此，水中飘着的塑料泡沫，山墙上的绿苔裂痕，以工笔描绘的华舍镇得以展现在眼前。

读《遍地枭雄》

发表于 2021-03-19 更新于 2021-03-27 分类于 杂谈 ， 读书笔记 阅读次数： Valine：
本文字数： 1.6k 阅读时长 ≈ 1 分钟

校历第二周。

开学季闲来没有什么事，读书的速度便快一些。不过近代史、C语言之类的课看书容易走神，本来半天读完的书，要读十几节课。淘宝上一本几块钱的书，买了王安忆的五本，可惜没有《长恨歌》。说实话，纸质的阅读体验确实要好很多。

这本书大致可以分为两个部分，主人公前面叫韩燕来后面叫毛豆。韩燕来在家里被姐姐宠的要死，没怎么接触社会，导致了他天真单纯的性格。在常人眼里日本工厂领着高工资，舒适的工作环境，他却由着性子辞掉了，商量着和朋友一同创业，在朋友们在现实的压迫下作鸟兽散以后，韩燕来又只能在家里待业，之后转去本土工厂上班。那时候上海想来也是一个发展极其迅速的城市，只要有心，什么工作都不差。可是韩燕来却恋家，表叔的木匠手艺也不愿意离家去学，怎么看都是个不会做打算的小孩子，工厂倒闭后众人去讨薪也就凑个热闹，似乎不在意薪水发不到自己头上。最后还是姐姐出钱让韩燕来学车，他才当上了出租车司机，又有能力养活自己。

读《欧也妮·葛朗台》

发表于 2021-03-10 更新于 2021-03-27 分类于 杂谈 ， 读书笔记 阅读次数： Valine：
本文字数： 1.3k 阅读时长 ≈ 1 分钟

开年来读的第一本书。寒假没日没夜地守在电脑旁边，说起来感觉也有些虚度，这本书本来计划在上学的火车上看的，不过车上睡过去了一半，醒着就刷手机，一页也没动。最后拖到了三月份才读完这本薄薄的书。

读完让人印象最深刻的是葛朗台老头这个标准的吝啬鬼。这种形象在巴尔扎克的小说中似乎屡见不鲜，他们将拥有的财产作为生命的价值。商人之间的利益联姻想起来，在中国古典名著中似乎没有类似的描写，而是政治联姻，可能是因为没有资本主义社会的财产丰富？抑或是封建制度中商人极低的地位。不过本质都是一样的，无论时间定格在哪一个年代，感情为财富服务的故事数不胜数。
